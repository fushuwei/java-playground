# MFA

多因子认证（MFA, 全称：Multi-Factor
Authentication）是一种安全机制，要求用户除了输入密码（第一因子）之外，还需提供至少一种完全不同类型的凭证（第二因子，如手机验证码、生物识别或安全密钥），才能成功登录账户，从而显著提升安全性，防止因密码泄露导致的数据泄露。

双因子认证（2FA, 全称：Two-Factor Authentication）是最常见的 MFA。
